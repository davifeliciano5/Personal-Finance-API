# 💰 Personal Finance API

API REST desenvolvida em Java com Spring Boot para gerenciamento de finanças pessoais.

## 🚀 Funcionalidades

* Cadastro e autenticação de usuários (JWT)
* Registro de receitas e despesas
* Categorias personalizadas
* Relatórios financeiros por período

## 🛠️ Tecnologias

* Java 17
* Spring Boot
* Spring Security
* JWT
* PostgreSQL
* Docker
* Swagger (OpenAPI)

## 📚 Documentação

Swagger disponível em:
`/swagger-ui.html`

## 🌐 API Online

👉 https://sua-api.onrender.com

## 📦 Rodar localmente

```bash
git clone https://github.com/seuuser/finance-api
cd finance-api
./mvnw spring-boot:run
```

## 🧪 Testes

```bash
./mvnw test
```
